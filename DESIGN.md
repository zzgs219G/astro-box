# Fuwari UI 设计理念

「极致克制的优雅流动感」 —— 用最少的视觉元素，创造最舒适、最高级的阅读与浏览体验。


## 技术栈与 UI 落地理念 (Tech Stack & Design Realization)

为了完美呈现“轻盈、丝滑、极简”的日系美学，本项目拒绝使用传统的粗暴 Vanilla JS 操纵，锁死以下现代化高能技术栈，将设计规范直接编织进代码底层：

### 1. 架构与性能：Astro (现代群岛架构)
*   **设计映射：** “隐形设计”与极致轻量。
*   **落地理念：** 全站内容在构建时直接液化为纯静态 HTML 运送给用户，去除一切不必要的运行时 (Runtime) 视觉负担。零首屏白屏，页面加载“触手即及”。

### 2. 页面内交互：Svelte (流体动力引擎)
*   **设计映射：** “消除视觉噪音”与内容至上。
*   **落地理念：** 拒绝任何突兀的硬编码显隐。所有的卡片过滤、标签切换、搜索筛选，全部交由 Svelte 的轻量级过渡动画引擎接管。利用 FLIP 机制实现卡片的流体重新排版，实现柔和、带呼吸感的无缝淡入淡出 (Fade-in/out)。

### 3. 跨页面转场：Swup (无感传送门)
*   **设计映射：** “丝滑的流动感”。
*   **落地理念：** 全面拦截浏览器传统的白屏刷新机制。当用户在未来扩展的子页面间跳转时，Swup 将整个主要内容区包裹进平滑的转场动画中（整体轻量淡出、新页柔和浮现），提供如同移动端原生 App 般的连贯视觉体验。

### 4. 视觉与色彩：Tailwind CSS (高级原子化软装)
*   **设计映射：** 单色调优雅 (HSL/OKLCH 动态主题) 与呼吸感留白。
*   **落地理念：** 彻底抛弃硬编码色值。利用 Tailwind 的原子化类名，全站深度定制基于 `--hue` 变量的 OKLCH 色彩空间。通过轻量投影、大面积毛玻璃材质 (`backdrop-blur`) 以及圆润药丸组件 (`rounded-full`)，营造精致的杂志级排版。

### 5. 数据约束：TypeScript (视觉安全监理)
*   **设计映射：** 数据驱动与结构稳定。
*   **落地理念：** 在代码编译层对 `Category`, `Tag`, `ResourceItem` 等核心视觉元素进行强类型约束，确保未来进行工具箱、聚合网等大规模内容扩建时，UI 表现依然绝对一致、绝不发生样式崩塌。

### 6. 隐形生态：Pagefind + Expressive Code + Lucide Icons
*   **设计映射：** 克制的细节与功能美学。
*   **落地理念：**
    *   **Pagefind：** 实现无需服务器的 0.001 秒本地离线全文检索，让搜索框交互灵敏深邃。
    *   **Expressive Code：** 赋予代码块 Mac 风三色圆点与高质感霓虹高亮，消除粗糙感。
    *   **Lucide Icons：** 全站图标统一锁死，确保线条粗细、几何比例高度一致，杜绝视觉混乱。

## 核心设计哲学

### 内容至上（Content First）
* 界面永远为内容服务，绝不抢戏。
* 大量留白（Generous Whitespace）让文字和图片自然呼吸。
* 所有装饰性元素都被极致简化，视觉焦点始终集中在文章本身。

### 丝滑的流动感（Fluid & Buttery Smooth）
* 利用 Svelte + Swup 技术引擎，实现页面内与页面间真正丝滑无缝的流动过渡。
* 所有交互（hover、点击、滚动、主题切换）都追求高级、轻盈、不卡顿的手感。
* 动画遵循「慢进快出」的自然规律，避免生硬的弹跳或过度效果。

### 现代极简主义 + 柔和日系美学
* 以柔和暗色模式为主，搭配可自定义的主题色（Hue 系统）。
* 圆角、阴影、间距都经过精心调校，营造高级感和舒适度，而非冰冷的工业风。
* 排版极致讲究：舒适的行高、清晰的层级、优雅的标题字体搭配。

### 隐形设计（Invisible Design）
* 好的 UI 应该是「感觉不到它的存在」。
* 导航栏、侧边栏、搜索、目录等功能都采用轻量化设计，在需要时出现，不需要时几乎隐形。
* 避免视觉噪音（No Visual Noise）：无过多分割线、无多余图标、无花哨特效。

## 设计原则细节

* **颜色系统**：基于 OKLCH 色彩空间 + CSS 变量 `--hue`，实现单色调优雅主题，支持亮/暗模式无缝切换。
* **排版系统**：正文追求极致阅读舒适度，代码使用 JetBrains Mono，标题采用柔和的层级对比。
* **交互反馈**：所有可点击元素都有细腻的 `hover` 和 `active` 状态，使用统一的 expand-animation 呼吸感。
* **性能优先**：即使动画丰富，也严格控制 GPU 消耗，保证在各种设备上都保持丝滑。
* **响应式**：从手机到大屏都保持一致的高级感，采用窄栏居中阅读布局。

## 一句话总结
Fuwari 的 UI 不是为了炫技，而是为了让作者的内容「安静而优雅地发光」。
它追求的是：打开即舒适、阅读即享受、浏览即愉悦的高级沉浸式体验。
