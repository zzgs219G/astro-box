# Fuwari UI 设计理念

「极致克制的优雅流动感」 —— 用最少的视觉元素，创造最舒适、最高级的阅读与浏览体验。

## 核心设计哲学

### 内容至上（Content First）
* 界面永远为内容服务，绝不抢戏。
* 大量留白（Generous Whitespace）让文字和图片自然呼吸。
* 所有装饰性元素都被极致简化，视觉焦点始终集中在文章本身。

### 丝滑的流动感（Fluid & Buttery Smooth）
* 采用 Swup + View Transitions 技术，实现页面间真正丝滑的无刷新过渡。
* 所有交互（hover、点击、滚动、主题切换）都追求高级、轻盈、不卡顿的手感。
* 动画遵循「慢进快出」的自然规律，避免生硬的弹跳或过度效果。

### 现代极简主义 + 柔和日系美学
* 以柔和暗色模式为主，搭配可自定义的主题色（Hue 系统）。
* 圆角、阴影、间距都经过精心调校，营造高级感和舒适度，而非冰冷的工业风。
* 排版极致讲究：舒适的行高、清晰的层级、优雅的标题字体搭配。

### 隐形设计（Invisible Design）
* 好的 UI 应该是「感觉不到它的存在」。
* 导航栏、侧边栏、搜索、目录等功能都采用轻量化设计，在需要时出现，不需要时几乎隐形。
* 避免视觉噪音（No Visual Noise）：无过多分割线、无多余图标、无花哨特效。

## 设计原则细节

* **颜色系统**：基于 OKLCH 色彩空间 + CSS 变量 `--hue`，实现单色调优雅主题，支持亮/暗模式无缝切换。
* **排版系统**：正文追求极致阅读舒适度，代码使用 JetBrains Mono，标题采用柔和的层级对比。
* **交互反馈**：所有可点击元素都有细腻的 `hover` 和 `active` 状态，使用统一的 expand-animation 呼吸感。
* **性能优先**：即使动画丰富，也严格控制 GPU 消耗，保证在各种设备上都保持丝滑。
* **响应式**：从手机到大屏都保持一致的高级感，采用窄栏居中阅读布局。

## 一句话总结
Fuwari 的 UI 不是为了炫技，而是为了让作者的内容「安静而优雅地发光」。
它追求的是：打开即舒适、阅读即享受、浏览即愉悦的高级沉浸式体验。
